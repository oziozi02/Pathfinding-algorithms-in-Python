# Pathfinding-algorithms-in-Python
This repository contains three pathfinding algorithms implemented as agents<br/>
for a game created in Python using the Pygame library.<br/>
More details about the game and the algorithms can be found in the .pdf [file](IS_DZ1_2025.pdf).
### Implemented algorithms
- Depth-first search
- Branch and bound
- A*

The implementation of the algorithms(agents) can be found in [agents.py](isdz1/core/agents.py).
### Requirements
- Pygame library(Can be installed via pip using ``` pip install pygame ```)
